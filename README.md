# API Backend - Interlink 

API RESTful desarrollada con Node.js, Express y MongoDB para la gestión de usuarios, autenticación y proyectos multidisciplinarios.

## Tecnologías Utilizadas
* Node.js & Express
* MongoDB & Mongoose
* JSON Web Tokens (JWT) & Bcrypt para seguridad
* Arquitectura en capas (Routes, Controllers, Services, Repositories)

2. Instalar las dependencias del proyecto:
